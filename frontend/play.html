How Sudoku Works

The goal is to fill every empty cell with a number from 1 to 9 so that each row, each column, and each 3×3 box contains all digits exactly once. You use logic to figure out which numbers can go in each spot based on what’s already filled in. No guessing is needed, every move should follow from the constraints.

Constraints
Each row, column, and 3×3 box needs digits 1–9 exactly once.
